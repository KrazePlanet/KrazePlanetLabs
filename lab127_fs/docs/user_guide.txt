FRIDA DATA PORTAL — USER GUIDE
Version 2.1 | Last Updated: March 2022
Classification: UNCLASSIFIED
=====================================

1. OVERVIEW
-----------
The FRIDA Data Portal provides authorized users with access to
calibration data, instrument reports, and annual summaries from
the Field Research Integrated Data Archive (FRIDA) program.

2. ACCESSING THE PORTAL
------------------------
The portal is accessible at: https://████.mil/
Authentication is required for restricted datasets. Public
calibration data and reports do not require authentication.

3. DOWNLOADING FILES
--------------------
Files are downloaded via the download.php endpoint:

  https://████.mil/download.php?filePathDownload={path}

Where {path} is the relative path to the file within the
authorized data directories:

  data_products/   — Calibration data, sensor outputs
  reports/         — Quarterly and annual reports
  docs/            — User guides and documentation

Example:
  download.php?filePathDownload=data_products/MISC/frida_cal/README.txt

4. SUPPORTED FILE TYPES
-----------------------
  .txt   Plain text documents
  .csv   Comma-separated calibration data
  .pdf   Reports (where available)
  .xml   Structured data exports

5. TROUBLESHOOTING
------------------
  "Access Denied" — Ensure the path begins with a valid directory
                    (data_products/, reports/, or docs/)
  "File not found" — Verify the exact filename and path
