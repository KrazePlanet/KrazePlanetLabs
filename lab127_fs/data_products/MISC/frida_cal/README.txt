FRIDA CALIBRATION DATASET — README
===================================
Dataset:    FRIDA Instrument Calibration Products
Version:    2022.Q2 (April–June 2022)
Owner:      Naval Research Laboratory, Measurement Division
Contact:    nrl-data@████.mil
Classification: UNCLASSIFIED // FOR OFFICIAL USE ONLY

DESCRIPTION
-----------
This dataset contains calibration files for the FRIDA (Field Research
Integrated Data Archive) environmental sensor array deployed across
DoD monitoring stations in the continental United States.

FILES INCLUDED
--------------
  cal_2022_Q2.csv       Quarterly calibration coefficients
  frida_report.txt      Instrument calibration report Q2 2022
  README.txt            This file

USAGE
-----
Calibration data is to be used by authorized personnel only.
All access is logged and audited per DoD Instruction 8500.01.

Authorized users may download files via:
  https://████.mil/download.php?filePathDownload=data_products/MISC/frida_cal/{filename}
